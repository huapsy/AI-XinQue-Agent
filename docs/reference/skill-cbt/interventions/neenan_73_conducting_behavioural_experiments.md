## 步骤

1. 选定一个最近发生的、足够小的具体情境。
2. 按技术步骤练习，并记录情绪强度（0–10）与关键想法。
3. 写下练习后的新发现（哪怕很小）。
4. 若痛苦骤升，暂停并改用稳定化策略或寻求支持。

- 自动化思维、回避、焦虑抑郁相关模式

**典型情境**
