# 引导式自我发现提问

## 简介

当情绪来了但说不清原因时，用开放式问题帮自己把「模糊感受」落到「具体想法」上，类似治疗中的 guided discovery / Socratic questioning，但由自己提问或由 AI 按此结构提问；重点是激发思考而非追求「标准答案」。

## 步骤

1. 先锚定一个具体时刻（何时、何地、当时在做什么）。
2. 用开放式问句问自己：「那时候脑子里闪过什么？」避免「你是不是觉得自己很差？」这类诱导句。
3. 若只能说「说不清」，再收窄：「最先注意到的是身体感觉、画面，还是一句话？」
4. 把出现的念头用原话写下，标成「待检验的想法」而不是事实。
5. 若连续卡住，可暂时停下或换更简单的情境练习，避免变成自我逼问。

## 应用场景与问题

**常见困扰**

- 情绪低落、烦躁、焦虑突然出现，但「找不到原因」
- 容易笼统地说「我就是这种人」「一切都很糟」

**相关情绪与症状**

- 抑郁心境、烦躁、紧张、空虚感

**典型情境**

- 独处时情绪反扑、睡前反刍、社交或工作后的情绪余波

## 适用与禁忌

- 危机或极度解离时不宜独自深挖，需先稳定与安全支持。
