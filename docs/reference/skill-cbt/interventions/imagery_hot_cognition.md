# 意象重现定位「热点想法」

## 简介

在难以直接说出自动化思维时，先在脑中用现在时重演当时场景，留意画面与身体感受，再把画面赋予的意义用语言说出来，从而接近情绪背后的想法（书中称 hot thoughts）。

## 步骤

1. 选一个引发困扰的具体场景（如「走在街上」「开会发言前」）。
2. 闭眼或注视固定点，用现在时描述：「我正在… 身体哪里紧/慌？」
3. 问：「若抬头/继续待在那个画面里，我会看到什么？」留意是否回避某些意象。
4. 再问：「这些画面对我来说意味着什么？」把答案写成短句（即候选 NAT）。
5. 记录 1～3 条最刺痛的句子，供后续与事实对照或认知练习使用。

## 应用场景与问题

**常见困扰**

- 「知道很焦虑但不知道自己在想什么」
- 社交场合身体不适明显但思维模糊

**相关情绪与症状**

- 社交焦虑、惊恐样紧张、羞耻感

**典型情境**

- 公共场所、人际互动、被注视或评价的情境

## 适用与禁忌

- 若意象引发强烈痛苦或闪回，应放慢节奏或寻求专业支持，不宜强行暴露。
