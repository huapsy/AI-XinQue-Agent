# 要点 36：IN- SESSION EMOTIONAL CHANGES

## 简介

要点 36：IN- SESSION EMOTIONAL CHANGES。以下为基于 Neenan《100 key points》要义的**用户向自助说明**，非原文。 语境摘录（自动截取）：「Changes in a client’s emotional state can occur at any point in each session. The therapist needs to be alert to these emo-tional shifts as they are important entry points into the client’s thinking. These shifts can be obvious (e.g. suddenly becoming angry) or subtle (e.g. narrowing eyes). The client may be talking about an issue in a non- emotive way when the ther-apist notices something in his manner that she infers is an emotional change: …」

## 步骤

1. 选定一个最近发生的、足够小的具体情境。
2. 按技术步骤练习，并记录情绪强度（0–10）与关键想法。
3. 写下练习后的新发现（哪怕很小）。
4. 若痛苦骤升，暂停并改用稳定化策略或寻求支持。

**典型情境**
