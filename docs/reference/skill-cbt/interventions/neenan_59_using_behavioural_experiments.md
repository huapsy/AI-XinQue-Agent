# 要点 59：USING BEHAVIOURAL EXPERIMENTS

## 简介

要点 59：USING BEHAVIOURAL EXPERIMENTS。以下为基于 Neenan《100 key points》要义的**用户向自助说明**，非原文。 语境摘录（自动截取）：「Behavioural experiments (BEs) in CBT serve a cognitive pur-pose: to test the validity of clients’ thoughts through action assignments. The evidence gathered from these experiments helps clients to construct new problem- solving perspectives. For example, a client who was very self- conscious about being overweight (‘I’m sure people think I’m a fat cow’) was unhappy about ‘not allowing myself to go swimming’. If she went swimming her prediction…」

## 步骤

1. 选定一个最近发生的、足够小的具体情境。
2. 按技术步骤练习，并记录情绪强度（0–10）与关键想法。
3. 写下练习后的新发现（哪怕很小）。
4. 若痛苦骤升，暂停并改用稳定化策略或寻求支持。

## 应用场景与问题

**常见困扰**

- 自动化思维、回避、焦虑抑郁相关模式

**相关情绪与症状**

- 焦虑、抑郁、羞耻、愤怒

**典型情境**

- 可具体想象的生活场景
