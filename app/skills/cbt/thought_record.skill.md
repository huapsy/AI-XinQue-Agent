## 目标
帮助用户系统性地记录触发情境、自动化思维、情绪反应，并尝试形成替代想法。这是认知重构的书面版本，适合用户需要更结构化的记录时使用。

## 引入话语
"你提到了好几个让你难受的想法。有一个方法可以帮你把这些想法理清楚——就像做一个小记录，把情境、想法和感受都写下来，然后看看有没有另一种看法。你愿意试试吗？"

## 执行流程

### 第1步：选择一个情境
"先想一个最近让你感受最强烈的情境，简单描述一下发生了什么。"
→ 引导用户聚焦一个具体事件。

### 第2步：记录自动化思维
"在那个情境中，你脑海里冒出来的第一个想法是什么？"
→ 如果用户给出多个想法，选最强烈的那个。

### 第3步：记录情绪和强度
"当这个想法出现的时候，你的感受是什么？强度有多高？比如0-10分。"

### 第4步：引导填写卡片
渲染想法记录卡片，帮用户结构化地写下情境、想法、情绪、证据和替代想法。

### 第5步：形成替代想法
"如果换一个角度来看这件事，有没有一种更完整、更平衡的说法？"
→ 不是否定原来的想法，而是扩展视角。

## 卡片内容

```json
{
  "type": "guided_exercise",
  "title": "想法记录",
  "description": "记录触发情境、自动思维、情绪，并探索替代想法",
  "steps": [
    {"instruction": "写下触发情境：发生了什么事？", "duration": 30},
    {"instruction": "写下自动思维：脑海里冒出来的第一个想法是什么？", "duration": 20},
    {"instruction": "写下情绪：你的感受是什么？强度0-10分？", "duration": 15},
    {"instruction": "支持证据：有哪些事实支持这个想法？", "duration": 20},
    {"instruction": "反对证据：有哪些事实不支持这个想法？", "duration": 20},
    {"instruction": "替代想法：一个更完整、更平衡的说法是什么？", "duration": 20}
  ],
  "rounds": 1
}
```

## 结束与评估
"你刚才做了一份完整的想法记录，把一个模糊的感受变成了一个可以看清楚的东西。当你看到替代想法的时候，情绪有变化吗？"

## 作业布置
"如果你愿意，这周遇到让你情绪波动比较大的时刻，可以用同样的方式记录一下：发生了什么、我在想什么、我的感受是什么。不需要每次都写替代想法，先记录就好。"

## 注意事项
- 想法记录比口头认知重构更结构化，适合喜欢条理清晰的用户
- 不要要求用户写太多条，一次只做一个情境
- 如果用户不确定情绪名称，可以提供选项（焦虑、愤怒、悲伤、委屈、无力…）
